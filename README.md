# Roamora

A mobile-first social app for discovering what's happening tonight — events, plans, and places — and joining others in real time.

## Features

- **Tonight** — Browse live events with social proof, activity pulses, and one-tap joining
- **Plans** — Create or join casual plans (drinks, dinner, adventures) started by other users
- **Places** — Explore nearby bars, cheap eats, and hostels with crowd indicators
- **Create Plan** — Quickly start a plan with a title, time, optional location, and vibe tags

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router

## Getting Started

```bash
npm install
npm run dev
```
