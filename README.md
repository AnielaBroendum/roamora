# Roamora

🌍 **[roamora.lovable.app](https://roamora.lovable.app)**

A mobile-first social app for travelers and locals to discover what's happening tonight — events, plans, and places — and connect with others in real time.

## Features

- **Tonight** — Browse live events with social proof, real-time activity pulses, and one-tap joining
- **Plans** — Create or join casual plans (drinks, dinner, adventures) started by other travelers
- **Places** — Explore nearby bars, cheap eats, and hostels with crowd indicators
- **Activity & Chat** — Chat threads for joined events with event imagery and real-time messaging
- **Profiles** — Set your display name, bio, nationality flag, and profile picture with crop/zoom support
- **Browse without sign-up** — Explore all content freely; sign up only when you want to join

## Tech Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS v3 + shadcn/ui
- React Router
- Supabase (Auth, Database, Storage)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.
